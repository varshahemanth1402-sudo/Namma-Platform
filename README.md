# ನಮ್ಮ ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ : Namma Platform

A simple Android app that helps Karnataka railway passengers find their train, platform, and coach — in Kannada and English.

Built for senior citizens and first-time travellers who need quick, clear information at the station.

---

## Features

- Search any Karnataka station in Kannada or English
- See the next 3 trains with platform numbers
- Scrollable color-coded coach strip for each train
- Text-to-Speech readout in Kannada or English
- One-tap language toggle

---

## Tech

Kotlin, XML, Android TTS API, local JSON data

---

## Run Locally

```bash
git clone https://github.com/justsakshi/NammaPlatform.git
```

Open in Android Studio and run on any device or emulator (API 21+).

For Kannada TTS: Settings → General Management → Language → add Kannada.
